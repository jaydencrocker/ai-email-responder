# AI Email Responder
This Python script automatically replies to business emails using keyword-based logic. It connects to an inbox via IMAP and sends replies using SMTP.

## Features
- Checks unread emails
- Parses subject/sender
- Sends automated responses

## Requirements
- Python 3
- IMAP + SMTP access
- Gmail (or any email service)

## How to Run
Edit the script with your email login. Run:
```bash
python email_responder.py
